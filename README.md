## ğŸ” How to Use / Run the Project

1. **Clone the repository**

   ```bash
   git clone https://github.com/JoyboyX69/E-Commerce-Sales-Analysis.git
   ```

2. **Import data into your SQL database**

   * Use tools like **MySQL Workbench**, **PostgreSQL**, **SQLite**, etc.
   * Import tables from CSV files provided in the `data/` folder.

3. **Run the SQL scripts**

   ```bash
   mysql -u username -p < sql_queries/01_load_data.sql
   mysql -u username -p < sql_queries/02_basic_analysis.sql
   ```

4. **View results**

   * Results are returned in the SQL console or exported to CSV files.
   * You can visualize results using tools like Excel, Power BI, Python, etc.
